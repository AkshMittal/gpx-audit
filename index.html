How to read the graphs

                
Interaction: Pan to inspect different ranges of the graph.

                
                
Time Delta KDE / Distance Delta KDE

                
Shows the distribution of consecutive time or distance deltas derived from the GPX

                
Peaks indicate commonly occurring delta ranges

                
Spread indicates variability in sampling behavior

                
Multiple peaks may indicate changes in sampling mode or logging behavior

                
This does not imply correctness or quality by itself

                
                
Log-scaled x-axis

                
The x-axis is log-scaled to expand detail near small deltas where most data lies

                
Values are shown in linear units, but spacing reflects logarithmic scaling

                
                
Rug plot

                
Each dash represents one observation

                
The rug shows where data exists, not how many values are identical

                
Dense or cluttered regions usually indicate many similar, not identical, values

                
For density and dominance, refer to the KDE curve

Time–Distance scatter plot

              
                
Each point represents one consecutive pair of GPX points.

                
The x-axis shows the time delta between points.

                
The y-axis shows the distance delta between the same points.

              
                
This plot is a global joint view.

                
It shows how time gaps and spatial movement relate across the entire file, without preserving sequence.

              
                
Common interpretations:

                
• Large time delta + small distance may indicate pauses or logging gaps

                
• Large distance + small time may indicate GPS jumps or teleportation

                
• Dense clouds near the origin often indicate regular sampling

              
                
Important limitation:

                
This plot does not preserve temporal order.

                
Points that are far apart in time may appear adjacent.

              
                
As a result, this view is best used to identify global regimes and extreme outliers, not local behavior.

Sequence-aware scatter plot

              
                
This plot preserves event order.

                
The x-axis represents GPX index (event sequence).

                
The y-axis represents time delta between consecutive points (log-scaled).

              
                
Each point still represents one consecutive point pair, but position now reflects where it occurred in the recording.

              
                
The color of each point encodes the corresponding distance delta.

                
Refer to the color legend to understand how distance varies over time.

              
                
This view is designed for local inspection.

                
It allows patterns to be read along the timeline rather than across the entire dataset.

              
                
What to look for:

                
• Extended horizontal bands indicate stable sampling intervals

                
• Vertical changes indicate sudden shifts in logging behavior

                
• Dense regions with low distance color may indicate stationary or low-activity logging

                
• Abrupt color changes may indicate transitions between movement states

              
                
This plot does not classify behavior.

                
It makes local structure visible so it can be judged in context.

            

            

                

                  The time–distance scatter and the sequence-aware scatter are complementary.
                  The former reveals global joint structure.
                  The latter reveals how that structure unfolds locally over time.
